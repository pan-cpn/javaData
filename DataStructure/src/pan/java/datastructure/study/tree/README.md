##树结构
##节点的度:便是节点拥有子树的颗数
##节点的层次:节点的层数从跟借点向下不断递增
##节点的高度:节点的最大深度
##节点的边:链接定两个节点的节点连接线
##节点的路径:需要查找根节点到指定节点的边数
##节点的直径:最后一个边缘节点到跟借点的最长路径
##无序树:节点之间没有次序
##有序树:节点之间有次序
##二叉树:二叉树表示每一个节点最多只有两个子杰点
##满二叉树:高度为h的具有2的n次方-1的树
##完全二叉树:里面的每一个捡点能够域满二叉树的每一个节点一一位置对应
##不完全二叉树:只有单边的节点的边的二叉树
##二叉树的遍历:
* 先序遍历:访问完跟节点后，遍历左边子树后在遍历右边子树
* 中序遍历:遍历左边子树后遍历后访问跟借点遍历右边节点
* 后序遍历:遍历左边子树后遍历右边子树，最后访问跟节点
